# Lottery Number Generator

## Overview

A simple web application to generate random lottery numbers.

## Project Outline

### Style and Design
- **Layout:** Centered layout with a clean and modern design.
- **Color Palette:**
  - Background: A subtle noise texture.
  - Numbers: Displayed in colored circles.
  - Button: A "glow" effect on hover.
- **Typography:** Expressive and easy-to-read fonts.
- **Visual Effects:** Multi-layered drop shadows for depth.

### Features
- **Lottery Number Generation:** Generates 6 unique random numbers between 1 and 45.
- **User Interface:**
  - A title for the application.
  - A container to display the generated lottery numbers.
  - A button to trigger the number generation.
- **Interactivity:** The button will have a hover effect.
